# Code of Conduct

## Our Commitment

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/).

This code of conduct applies to all project spaces, including:

- GitHub repositories and issue trackers
- Pull requests and code reviews
- Project discussions and forums
- Project-related social media
- Project events and meetups

## Standards

We are committed to providing a welcoming and inclusive environment for all contributors, regardless of:

- Age, body size, disability, ethnicity, gender identity and expression
- Level of experience, education, or socio-economic status
- Nationality, personal appearance, race, religion, or sexual identity and orientation

## Reporting

If you experience or witness unacceptable behavior, or have any other concerns, please report it by contacting:

- **Primary Contact**: {{cookiecutter.author_email}}

All reports will be handled with discretion and confidentiality.

## Enforcement

Project maintainers are responsible for clarifying and enforcing our standards of acceptable behavior and will take appropriate and fair corrective action in response to any behavior that they deem inappropriate, threatening, offensive, or harmful.
